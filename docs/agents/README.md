## Suggested Subagent Pairings

- Major UI workflow change: UI/UX Agent + QA / Validation Agent.
- Upload/extraction/sectioning change: Backend Agent + QA / Validation Agent.
- Tag list, section taxonomy, scoring, or underwriting language change: Credit Domain Agent + UI/UX Agent.
- Export shape or training-file change: Backend Agent + QA / Validation Agent.
- Setup, launcher, install, README, GitHub packaging change: Release / Packaging Agent + QA / Validation Agent.
